## 📋 **Step-by-Step Deployment**

### **Step 1: Create Account (2 minutes)**

1. Go to: **https://www.000webhost.com/**
2. Click **"Free Sign Up"** (top right)
3. Fill in:
   - Email: (your email)
   - Password: (create strong password)
   - Website name: Choose a name (e.g., `adityamohan` or `techvision`)
   - Click "Sign Up"
4. Verify your email (check inbox)

### **Step 2: Create Website (1 minute)**

1. After login, click **"Create Website"**
2. Choose a website name (this becomes your URL):
   - Example: `adityamohan` → `adityamohan.000webhostapp.com`
   - Or: `techvision` → `techvision.000webhostapp.com`
3. Click "Create"
4. Wait for website to be created (~30 seconds)

### **Step 3: Upload Files (5 minutes)**

1. In your website dashboard, click **"File Manager"**
2. Navigate to **`public_html`** folder
3. **DELETE** the default files (index.html, etc.)
4. Click **"Upload Files"** button (top right)
5. Upload ALL these files from your portfolio:
   ```
   ✅ index.html
   ✅ style.css
   ✅ main.js
   ✅ contact.php
   ✅ favicon.svg
   ```
6. Wait for upload to complete

### **Step 4: Test Your Website! 🎉**

1. Visit: **`http://yourwebsitename.000webhostapp.com`**
2. Your portfolio should be live!
3. Test the contact form:
   - Scroll to contact section
   - Fill in all fields
   - Click "Send Message"

## 🆘 **Troubleshooting:**

**Contact form not working?**
- Check if all files uploaded correctly
- Verify `contact.php` is in `public_html` folder
- Test by filling the form and checking email

**Emails not arriving?**
- Check spam folder
- Some hosts have email delays (5-10 minutes)
- If still not working, let me know (I can set up SMTP)

**Website not loading?**
- Clear browser cache
- Wait 5-10 minutes for DNS propagation
- Try incognito/private browsing
