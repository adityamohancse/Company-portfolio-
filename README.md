# 🚀 Company Portfolio Website

A modern and responsive **Company Portfolio Website** built using HTML, CSS, JavaScript, and PHP.
Designed to showcase services, projects, testimonials, and contact functionality with a clean UI.

---

## 🌐 Live Demo

👉 [https://adityamohancse.github.io/Company-portfolio-/](https://adityamohancse.github.io/Company-portfolio-/)

---

## 📌 Features

* ✅ Responsive Design (Mobile + Desktop)
* ✅ Clean UI/UX Layout
* ✅ Modular Components (Navbar & Footer)
* ✅ Services & Projects Showcase
* ✅ Testimonials Section
* ✅ Contact Form (Backend Integration with PHP)
* ✅ Organized Folder Structure

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/adityamohancse/Company-portfolio-.git
```

2. Navigate to project:

```bash
cd Company-portfolio-
```

3. Open `index.html` in browser
   OR use Live Server in VS Code

---

## 📬 Contact Form

* Backend handled using **PHP**
* Located inside `/backend/contact.php`
* Can be extended with email APIs (like SMTP, SendGrid)
