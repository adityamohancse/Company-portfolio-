# TechVision Portfolio

Modern, responsive portfolio website built with HTML, CSS, and JavaScript featuring smooth animations, particle effects, and a working contact form.

## 🌟 Features

- ✨ Modern 2026 design with gradient effects
- 🎨 Animated particle background
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎯 Smooth scroll animations (AOS)
- 💼 Project showcase with filtering
- 💬 Testimonials slider
- 📧 Working contact form (Web3Forms)
- 🔗 Social media integration
- ⚡ Fast loading & optimized

## 🚀 Quick Start

### View Live Demo
Visit: [https://adityamohancse.github.io/Company-portfolio-/]

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/adityamohancse/portfolio.git
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using PHP
   php -S localhost:8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

3. Visit `http://localhost:8000`

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with custom properties
- **JavaScript (ES6+)** - Interactive features
- **Particles.js** - Animated background
- **AOS (Animate On Scroll)** - Scroll animations
- **Font Awesome** - Icons
- **Web3Forms** - Contact form handling

## 📧 Contact Form

The contact form uses [Web3Forms](https://web3forms.com/) - a free form handling service.

To use your own email:
1. Get free access key from: https://web3forms.com/
2. Replace the access key in `index.html` (line ~590)

## 🎨 Customization

### Colors
Edit CSS custom properties in `style.css`:
```css
--accent-primary: #6366f1;
--accent-secondary: #8b5cf6;
--text-primary: #1e293b;
```

### Content
- Update text in `index.html`
- Modify projects, services, and testimonials sections
- Change social media links
